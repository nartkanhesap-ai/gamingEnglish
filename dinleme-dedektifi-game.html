# Gaming English — Dinamik Kelime Sistemi Kurulum Kılavuzu ## Mimari ``` English Coach Firebase (users koleksiyonu) └── users/{uid}.level = "A1" | "A2" | "B1" | "B2" | "C1" Gaming English Firebase (sharedWords koleksiyonu) └── sharedWords/{docId} ├── game: "kelime-avi" | "gramer-quest" | "dinleme-dedektifi" | "quiz-hizli" | "yazi-sahnesi" ├── level: "A1" | "A2" | "B1" | "B2" | "C1" ├── month: "2025-01" ← YYYY-MM formatı ├── items: [ ... ] ← oyuna özgü veri dizisi └── updatedAt: Timestamp Cloudflare Worker (words-api.nartkanhesap.workers.dev) GET /words?game=kelime-avi&level=A1&month=2025-01 → sharedWords'dan ilgili items[] döner ``` --- ## Adım 1 — Firebase'de sharedWords koleksiyonu oluştur Firebase Console → Firestore → Yeni koleksiyon: `sharedWords` ### Firestore Security Rules (firestore.rules): ``` rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { // Öğrenciler sadece kendi profilini okuyabilir match /users/{uid} { allow read: if request.auth.uid == uid; allow write: if request.auth.uid == uid; } // sharedWords: herkes okuyabilir (Gaming English public site) // sadece admin yazabilir match /sharedWords/{docId} { allow read: if true; allow write: if request.auth.token.admin == true; } } } ``` --- ## Adım 2 — Cloudflare Worker'ı deploy et `word-system/cloudflare-worker.js` dosyasını Cloudflare Dashboard'dan deploy et: 1. workers.dev → Create Worker → "words-api" adını ver 2. Kodu yapıştır 3. Environment Variables ekle: - `FIREBASE_PROJECT_ID` = "english-coaching-2eeee" - `FIREBASE_API_KEY` = Firebase API key'in Worker URL: `https://words-api.nartkanhesap.workers.dev` --- ## Adım 3 — Aylık kelime güncelleme `word-system/update-words.js` script'ini her ay çalıştır: ```bash # Node.js gerekli npm install @anthropic/sdk firebase-admin # .env dosyası oluştur ANTHROPIC_API_KEY=sk-ant-... FIREBASE_SERVICE_ACCOUNT_PATH=./serviceAccountKey.json FIREBASE_PROJECT_ID=english-coaching-2eeee # Çalıştır node update-words.js --month=2025-02 ``` Script otomatik olarak: 1. Claude API'den her oyun × her seviye için yeni kelimeler/sorular üretir 2. Firestore'daki sharedWords koleksiyonuna kaydeder 3. Eski ayın verilerini arşivler (silmez) --- ## Adım 4 — Gaming English oyunlarına Firebase bağla Her oyun HTML'ine aşağıdaki config'i ekle (zaten ekli gelir): ```javascript const WORDS_API = 'https://words-api.nartkanhesap.workers.dev'; const ENGLISH_COACH_FIREBASE = { apiKey: "...", projectId: "english-coaching-2eeee" }; ``` --- ## Seviye → Oyun Zorluk Eşlemesi | English Coach Seviyesi | oyun easy | oyun medium | oyun hard | |------------------------|-----------|-------------|-----------| | A1 | ✅ aktif | ❌ kilitli | ❌ kilitli | | A2 | ✅ aktif | ✅ aktif | ❌ kilitli | | B1 | ✅ aktif | ✅ aktif | ✅ aktif | | B2+ | ✅ aktif | ✅ aktif | ✅ aktif | --- ## Firestore Döküman Örneği ```json { "game": "kelime-avi", "level": "A1", "month": "2025-01", "updatedAt": "2025-01-01T00:00:00Z", "items": [ { "word": "APPLE", "image": "🍎", "label": "Apple" }, { "word": "CAT", "image": "🐱", "label": "Cat" }, { "word": "HOUSE", "image": "🏠", "label": "House" } ] } ```
